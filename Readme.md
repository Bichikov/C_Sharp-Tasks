# Синтаксис C# 

## 1. Как рабоать с dotnet : 
``` 
1. dotnet new console - создать новый файл.

2. dotnet new gitignore - создавть файл gitignore.

3. dotnet run - запустить проект.
```

## 2. Вывод или ввод чего-то :
```
1. Console.Write(""); - Вывод в одну строку.

2. Console.WriteLine(""); - В конце перейти на новую строку(следующее предложеник будет с новой строки).

3. Console.ReadLine() - Cчитать строку из терминала;

4. Console.Write(number + " "); - Добавление пробела после числ.
```

## 3. Типы данных и типы операций :
```
Типы данных :
1. string - строка.

2. int - целые числа.

3. double - вещественные числа. 

4. bool - логический тип данных.(true or false)

Пример: int a = 3;
```
```
Типы операций: 
1. % - остаток от деления.

2. / - деление без остатка для int(для double остаток будет показывать).
```
